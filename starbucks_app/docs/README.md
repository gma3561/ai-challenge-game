# 그린빈 커피 PWA 데모 디자인

스타벅스 스타일의 커피 주문 앱 데모를 위한 UI/UX 디자인 구현입니다.

## 구현 내용

본 데모는 PRD 요구사항에 맞춰 다음과 같은 화면과 기능을 포함합니다:

1. **홈 화면**
   - 매장 정보 (강남역점, 영업시간)
   - 인기 메뉴 3종 (카페 라떼, 자바칩 프라푸치노, 얼그레이)
   - 전체 메뉴 보기 버튼

2. **메뉴 목록 화면**
   - 카테고리별 필터링 (커피, 티, 프라푸치노)
   - 각 메뉴 아이템은 이미지, 이름, 가격, 카페인/알레르기 배지 표시

3. **메뉴 상세/옵션 화면**
   - 메뉴 이미지, 이름, 기본 가격, 배지 표시
   - 옵션 그룹 (사이즈, 온도, 샷 추가, 우유, 시럽, 얼음, 당도)
   - 실시간 가격 계산
   - 장바구니 추가 버튼

4. **장바구니 화면**
   - 장바구니 아이템 목록 (이름, 옵션, 가격, 수량)
   - 수량 조절 및 삭제 기능
   - 합계 금액 계산
   - 결제하기 버튼

5. **결제 화면**
   - 주문자 정보 입력
   - 카드 정보 입력 (번호, 유효기간, CVC)
   - 결제 금액 표시
   - 데모용 카드번호 안내 문구

6. **주문 완료 화면**
   - 주문 완료 메시지
   - 주문번호 표시
   - 픽업 안내 문구
   - 다시 주문하기 버튼

## 기술 구현

- **반응형 디자인**: 모바일 우선(360-430px)으로 설계, 태블릿에서도 적절하게 표시
- **PWA 기능**: 웹 매니페스트, 서비스 워커, 오프라인 모드 안내
- **접근성**: WCAG 기본 준수 (적절한 대비, 레이블, 키보드 포커스 지원)
- **상태 관리**: 로컬 스토리지를 활용한 장바구니 유지
- **데이터 모델**: PRD 명세에 맞게 메뉴, 옵션, 장바구니, 주문 데이터 구조화

## 파일 구조

- `index.html`: 메인 HTML 파일
- `styles.css`: 스타일시트
- `app.js`: 자바스크립트 기능
- `manifest.webmanifest`: PWA 매니페스트
- `service-worker.js`: 서비스 워커
- `offline.html`: 오프라인 페이지
- `images/`: 이미지 디렉토리
  - `logo.svg`: 브랜드 로고
  - `icons/`: PWA 아이콘

## 실행 방법

1. 웹 서버에 파일들을 업로드하거나 로컬 서버를 실행합니다.
2. 브라우저에서 `index.html`을 엽니다.
3. 모바일 기기에서는 PWA로 설치할 수 있습니다.

## 테스트 시나리오

1. 홈 화면에서 인기 메뉴 중 하나를 선택하거나 전체 메뉴 보기
2. 메뉴 목록에서 카테고리별 필터링 및 메뉴 선택
3. 메뉴 상세 화면에서 옵션 선택 및 장바구니 추가
4. 장바구니에서 수량 조절 및 결제하기
5. 결제 화면에서 정보 입력 및 결제 (성공: 4242..., 실패: 4000...)
6. 주문 완료 화면 확인 및 다시 주문하기
